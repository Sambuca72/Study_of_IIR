# Sine Sum Calculation

Программа заполняет массив значениями синуса на одном периоде и вычисляет их сумму.

## Сборка
Для выбора типа данных используйте флаг `USE_DOUBLE`:

- **float (по умолчанию):**
```
  cmake -B build -DUSE_DOUBLE=OFF -S .
  cmake --build build
  ./build/main
```
- **double:**
```
  cmake -B build -DUSE_DOUBLE=ON -S .
  cmake --build build
  ./build/main
```
## Результаты
| Тип | Сумма |
| :--- | :--- |
| float | -0.0277862 |
| double | 6.27585e-10 |

Разница обусловлена тем, что `double` имеет вдвое большую разрядную сетку, что позволяет точнее представлять числа и минимизировать ошибки округления при выполнении 10 миллионов операций сложения.
